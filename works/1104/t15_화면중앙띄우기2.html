화면중앙에 새창띄우기(부모창)

텍스트 박스에 윈도우 가로크기와 세로크기를 입력받아서 버튼을 클릭하면 화면중앙에 사용자가 지정한 크기로 새창띄우시오
단, 최소 창의 크기는 (300px * 200px)

윈도우 가로크기 :

윈도우 세로크기 : 

        : 새창에 사용되는 이름(name)?
            _blank : 새창(기본값), _parent : 부모창에서 열림, _top : 로드된 프레임셋을 다시 대체함, _self : 자신창에 띄운다 
        : 새창에 사용되는 옵션(option)?
            width : 창의 폭
            left : x축의 위치
            top : y축의 위치
            fullscreen(yes/no/auto) : 전체화면모드(지원X)   // '예) fullscreen = yes'  , auto가 기본값
            titlebar(yes/no/auto) : 제목표시줄의 표시여부 결정
            location(yes/no/auto) : 주소표시줄의 출력여부를 결정
            menubar(yes/no/auto) : 메뉴표시줄의 표시여부를 결정
            resizable(yes/no/auto) : 창의 크기를 제어 결정
            scrollbars(yes/no/auto) : 스크롤바의 표시여부 결정 
            toolbar (yes/no/auto) : 툴바(도구상자)의 표시여부 결정
